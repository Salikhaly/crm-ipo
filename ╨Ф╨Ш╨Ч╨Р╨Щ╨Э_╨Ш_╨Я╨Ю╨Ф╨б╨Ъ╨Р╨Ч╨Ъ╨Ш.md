# 🎨 Дизайн v2 + подсказки для менеджеров

## Что изменилось визуально
- **Глубина вместо плоскости** — карточки приподнимаются при наведении, у кнопок мягкие тени, живой ховер.
- **Читаемость** — улучшена типографика (шрифтовые фичи Inter, трекинг), крупнее метрики.
- **Канбан** — карточки реагируют на наведение, скруглённее, приятнее таскать.
- **Плавность** — контент раздела мягко проявляется, переходы сглажены.

## Подсказки для менеджера (чтобы понял любой)
Новые визуальные компоненты (в globals.css, переиспользуются везде):
- `.hint` — голубой баннер с лампочкой 💡 «что тут делать»
- `.hint-warn` / `.hint-success` — жёлтый/зелёный варианты
- `.help-tip` — значок «?» с всплывающей подсказкой при наведении
- `.next-step` — акцентная карточка «следующий шаг»
- `.empty-state` — понятная заглушка на пустых экранах
- `.section-title` — единый стиль заголовков секций

## Где добавлены подсказки
| Экран | Подсказка |
|-------|-----------|
| Дашборд | «С этого начните день. Кликните клиента — откроется карточка...» над блоком «Требует внимания» |
| Калькулятор | «Как считать: По цене — платёж под квартиру, По зарплате — что потянет клиент...» |
| Карточка клиента | «Следующий шаг» — акцентная карточка, что делать по этапу (клик → нужная вкладка) |
| Новый клиент | «Заполните ФИО и телефон — остальное позже. Телефон в любом формате...» |
| KPI / Воронка | «Красный процент — там теряется половина клиентов, узкое место» |
| Пустые экраны | Крупные понятные заглушки с инструкцией |

## Принцип
Каждый экран теперь отвечает на три вопроса новичка:
**что это? что делать? куда нажать?** — без обучения и мануалов.
