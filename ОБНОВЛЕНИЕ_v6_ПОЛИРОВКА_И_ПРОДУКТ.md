## Волна 4 — «Полировка»

1. **Дизайн-токены** — палитра/радиусы/тени как CSS-переменные в `styles/globals.css` (`--brand`, `--ink`, `--r-md`…). Новый код пишем на токенах.
2. **Калькулятор в фирменной синей гамме** — раньше был зелёный, как отдельный продукт. Зелёный остался только семантически («одобрено», хороший КД).
3. **Иконки Tabler во вкладках карточки** вместо эмодзи — солиднее.
4. **Минимальный шрифт поднят** — подписи этапов 8.5px → 10px (мобильный 7.5 → 9).

## Волна 5 — «Продукт»

5. **Онбординг-тур первого входа** — 4 шага (канбан → карточка → WhatsApp → быстрые приёмы). Показывается один раз (localStorage `crm_tour_done`). Чтобы показать заново — удалить этот ключ в браузере.
6. **Экспорт CSV** — раздел «Поиск/Все клиенты», кнопка «Экспорт» (видят только руководитель и техник). Выгружает текущий отфильтрованный список: ФИО, телефон, ИИН, этап, менеджер, источник, договор, суммы, причина закрытия. Открывается в Excel (BOM + `;`).
7. **Конструктор авто-задач этапов** — Панель техника → Воронка: у каждого этапа поле «Авто-задача» (тип + текст). Пусто = авто-задачи нет. Не настроено = дефолт из кода, как раньше. *(нужна миграция 011)*
8. **Round-robin автораспределение WhatsApp-лидов** — Панель техника → WA-ответы → тумблер «Автораспределение лидов». Новый лид назначается менеджеру с наименьшим числом активных чатов — и чат, и клиент. *(нужна миграция 011; при выключенном тумблере поведение прежнее)*

## Деплой

1. **Supabase → миграция `011_auto_tasks_and_round_robin.sql`** (см. выше).
2. git add . && git commit -m "feat: v6 — design polish, tour, csv export, auto-task builder, round-robin" && git push
3. Проверить после деплоя:
   - калькулятор — синие акценты вместо зелёных
   - первый вход в анонимном окне → тур из 4 шагов
   - Поиск (под руководителем) → «Экспорт» → CSV открывается в Excel с кириллицей
   - Панель техника → Воронка → у этапов поля «Авто-задача» → изменить, сохранить, перенести клиента на этап → создалась настроенная задача
   - Панель техника → WA-ответы → включить «Автораспределение» → написать с нового номера в WA → лид получил менеджера
