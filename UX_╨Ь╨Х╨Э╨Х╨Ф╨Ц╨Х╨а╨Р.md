# 👩‍💼 UX для менеджера — 10 улучшений

## Что теперь умеет CRM

### 1. 👉 «Следующий шаг» в карточке
Открыл клиента — сверху синий блок: что делать прямо сейчас (по этапу воронки).
«Связаться за 15 минут», «Запросить документы»... Клик по блоку — открывает нужную вкладку.

### 2. 🔥 Дашборд «Требует внимания сегодня»
Три списка с утра: 🔴 новые лиды без ответа · 🟡 не трогали 3+ дней · 📞 задачи на сегодня.
Клик по имени — открылась карточка. Если всё чисто — «✅ Всё под контролем».

### 3. 📞💬 Позвонить/WhatsApp одним тапом
- В карточке клиента: кнопки «Позвонить» (набирает с телефона) и «WhatsApp»
  (открывает встроенный чат CRM; если чата нет — внешний WhatsApp)
- В канбане на каждой карточке: иконка трубки — сразу звонок

### 4. ⚠️ Защита от дублей
Создаёшь клиента, вводишь телефон — если номер уже есть, красное предупреждение:
«Этот номер уже есть: Иванов А. (менеджер: Асель)». Два менеджера больше
не позвонят одному клиенту.

### 5. 🔍 Поиск понимает любой формат номера
«8707 123-45-67», «+7 707 1234567», «7071234567» — всё находит одного клиента.
(Требуется миграция 009 — нормализует старые номера в базе.)

### 6. ⚡ Статус звонка одним тапом из канбана
На карточке в канбане: 📵 «не дозвонился» (ставит статус) и ⏰ «перезвонить»
(ставит статус + создаёт задачу на завтра). Без открытия карточки.

### 7. ❓ Словарик терминов в калькуляторе
Сворачиваемый блок «Что значат термины»: ПВ, КД, ПМ, аннуитет, ОПВ, ГЭСВ —
простыми словами для новичков.

### 8. 📋 Список документов одной кнопкой
В карточке клиента кнопка «Документы» — копирует готовый список документов
для ипотеки. Вставил клиенту в чат — ничего не забыл.

### 9. 🔔 Звук + бейдж на новые сообщения WA
Пришло сообщение — короткий звуковой сигнал (даже в активной вкладке)
+ бейдж с числом на пункте WhatsApp внизу. Push-уведомление при скрытой вкладке было и раньше.

### 10. 🗂️ Понятный пустой экран
Новый менеджер видит не пустую сетку, а «Пока нет клиентов. Нажмите + чтобы добавить»
с объяснением что лиды из WhatsApp создаются сами.
