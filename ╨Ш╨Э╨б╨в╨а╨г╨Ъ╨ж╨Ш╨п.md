# 🏠 ИПОТЕКА CRM v2 — Инструкция деплоя

## Что это

Next.js приложение. Фронтенд + бэкенд в одном проекте.
Деплоится на **Vercel** (бесплатно). База данных — **Supabase** (бесплатно).

---


## ШАГ 1 — Supabase (10 мин)

1. **supabase.com** → Sign up → New project
2. Settings → SQL Editor → New query
3. Вставьте содержимое `schema.sql` → Run
4. Settings → API → скопируйте:
   - `Project URL`
   - `service_role` ключ (не anon!)

---

## ШАГ 2 — GitHub (5 мин)

1. **github.com** → New repository → `mortgage-crm`
2. Нажмите "uploading an existing file"
3. Перетащите ВСЕ файлы из папки `crm-final`
4. Commit changes

---

## ШАГ 3 — Vercel (5 мин)

1. **vercel.com** → Add New Project → Import `mortgage-crm`
2. Перед деплоем добавьте переменные в "Environment Variables":

| Ключ | Значение |
|------|----------|
| `SUPABASE_URL` | https://xxx.supabase.co |
| `SUPABASE_SERVICE_KEY` | eyJ... (service_role) |
| `JWT_SECRET` | любая строка 32+ символов |
| `GREEN_API_ID` | пока оставьте пустым |
| `GREEN_API_TOKEN` | пока оставьте пустым |

3. Deploy → через 2 мин получите ссылку

---

## ШАГ 4 — Первый вход

Откройте ссылку → войдите: `admin` / `admin123`

Зайдите в **Панель техника** → Пользователи → поменяйте все пароли!

---

## ШАГ 5 — WhatsApp (после основного деплоя)

1. **green-api.com** → Регистрация → создать инстанцию
2. Отсканируйте QR своим WhatsApp (работает с обычным номером)
3. Скопируйте `idInstance` и `apiTokenInstance`
4. В Vercel → Settings → Environment Variables:
   - `GREEN_API_ID` = ваш idInstance
   - `GREEN_API_TOKEN` = ваш apiTokenInstance
5. В Green API → Settings → Webhook URL:
   ```
   https://ВАШ-САЙТ.vercel.app/api/wa/webhook
   ```
6. Включите: `Incoming messages` ✅ → Save

Теперь когда клиент пишет в WhatsApp:
- Автоматически создаётся лид в CRM
- Видите переписку в разделе WhatsApp
- Отвечаете прямо из CRM
- Фото, голосовые, видео, документы — всё работает
