## Что добавлено

### 1. Таблица клиентов + массовые действия
- Раздел «Поиск/Все клиенты» → переключатель **Карточки / Таблица** (запоминается).
- Таблица: сортировка кликом по любой колонке (клиент, телефон, этап, менеджер, источник, сумма, дата).
- **Массовые действия** (только руководитель/техник): чекбоксы → панель «Выбрано N» →
  сменить этап (кроме «Закрыто» — оно требует причину), назначить менеджера, экспорт выбранных в CSV.

### 2. Теги клиентов *(миграция 012)*
- В шапке карточки клиента: чипы `#тег`, «+ тег» → ввод + Enter, крестик удаляет.
- В списке: строка фильтра по всем существующим тегам, теги видны в таблице.
- Примеры использования: `vip`, `горячий`, `повторный`, `реклама-март`.

### 3. Центр уведомлений 🔔
- Колокольчик в топбаре на всех экранах с общим счётчиком.
- Внутри: 🔴 просроченные задачи (клик — карточка клиента), 🆕 новые лиды, WhatsApp непрочитанные (клик — раздел WA).

### 4. Календарь задач
- Раздел «Задачи» → переключатель **Список / Календарь**.
- Месячная сетка: число задач на день (красное — просрочка), клик по дню — список задач,
  клик по задаче — карточка клиента. Навигация по месяцам + «Сегодня».
