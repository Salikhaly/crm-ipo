# 🚀 PRODUCTION — финальный деплой Mortgage CRM

Все пункты аудита закрыты. Проверено: Next 14.2.33, сборка ✓, 167/167 тестов,
системная undefined-проверка ✅ по всем модулям.

## Что вошло в этот релиз
1. **Next.js 14.2.3 → 14.2.33** — закрыта critical-уязвимость (CVE middleware bypass
   и др. в патч-диапазоне). Без breaking changes, сборка и тесты проверены.
2. **Ключи программ унифицированы** — API-Ипотека и карточка клиента используют
   API_PROGRAMS_FALLBACK с ключами движка (nauryz20, jasyl...). Мина «Программа
   не найдена» при пустой БД обезврежена.
3. **fmtN — единый источник** (lib/constants.js), 3 локальные копии убраны.
4. Всё прежнее: разбивка на модули, калькуляторы из БД, WA anti-ban,
   security headers, RLS, CI.

## Оставшиеся npm-уязвимости (осознанно НЕ фиксим)
| Пакет | Severity | Почему не фиксим |
|-------|----------|------------------|
| next (image cache) | high | next/image НЕ используется в проекте — не применимо. Фикс только в Next 16 (breaking) |
| postcss | moderate | build-time only, не попадает в runtime |
| uuid (googleapis) | moderate | транзитивная, экзотический вектор (buf в v3/v5/v6) |
