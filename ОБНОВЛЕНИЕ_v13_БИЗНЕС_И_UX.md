# ➕ Дополнение 13.07: предрелизная волна аудит-фиксов (в этой же ветке)

Перед деплоем прогнаны два агента-аудитора (баги/бизнес-логика + UX) и code-review
диффа. Всё критичное исправлено и покрыто ритуалом (build ✓, jest 267/267 ✓).

## Безопасность и данные (сервер)
- **WA-вебхук**: клиент больше НЕ создаётся через upsert по phone (без unique
  constraint upsert падал → новые WhatsApp-лиды не создавали клиента; с constraint —
  затирал бы существующую карточку). Теперь: поиск по хвосту номера → insert только
  если не найден; существующая карточка не трогается.
- **XSS в «Скачать PDF» расчёта**: ФИО/имена (приходят из WhatsApp/Excel/ПКБ)
  экранируются перед document.write.
- **Роли на сервере**: менеджер не может переписать ответственного/ипотечного
  специалиста чужого клиента и взять ничейного может только себе; qa — только
  просмотр (раньше qa мог удалять клиентов!); DELETE — только admin/head (whitelist).
- **Слияние дублей**: сначала обновляется главный, потом удаляется дубль — ошибка
  посреди слияния больше не теряет данные.
- **Дедуп на сервере**: POST /api/clients отклоняет дубль по телефону (хвост 10 цифр)
  и ИИН по ВСЕЙ базе — двое менеджеров больше не заведут одного клиента.
- Поиск: запятые/скобки в запросе больше не дают 500; калькулятор не падает от
  программы без вариантов/с coeff=0/mrp=0; имя менеджера с кавычкой не ломает Drive;
  ПКБ-интеграция находит клиента и по ненормализованному номеру.

## Удобство (фронт)
- **Системная «Назад» на телефоне закрывает карточку клиента** (раньше — ничего/выход из CRM).
- **Диалог выхода**: третья кнопка «Сохранить и выйти» (раньше был тупик — либо сидеть, либо потерять).
- **Автосейв больше не молчит**: если карточка не сохраняется (ИИН, дубль телефона) — причина показывается тостом.
- **Задачи**: кружок слева — теперь настоящий чекбокс (клик закрывает задачу из списка); появилась кнопка «+ Задача» с формой.
- **WA быстрый расчёт**: ошибки показываются тостом, а не подставляются в поле отправки (риск отправить клиенту «❌ Ошибка…»).
- Разрешён pinch-zoom; «Готово» импорта жёлтое, если создано 0; кнопка «Новый клиент» снова с подписью; тексты «через Поиск» → «Список клиентов»; URL вебхука в инструкции WA — текущий домен.
- Лендинг /promo: номер WhatsApp берётся из env **NEXT_PUBLIC_WA_PHONE** — задайте его в Vercel, иначе заявки уходят на заглушку!
- Маршруты: этап с пустым названием больше не роняет карточку и не сохраняется из админки.
