# 📈 МАРКЕТИНГ — Ипотека CRM

## Позиционирование (одно предложение)

**«Операционка ипотечного брокера в Казахстане: лид из WhatsApp → расчёт по госпрограммам → договор → сопровождение до ключей — в одной системе».**

Не «калькулятор для одного специалиста», а рабочая система для всей команды:
менеджер продаёт, специалист ведёт сделку, руководитель видит цифры, техник настраивает — все в одном окне.

## Почему мы, а не amoCRM / Bitrix / Excel

| Боль брокера | Что у нас |
|---|---|
| amoCRM ничего не знает про ипотеку РК | Госпрограммы (Наурыз, 7-20-25, Отбасы, 50/50, Аскери...) зашиты в калькулятор, ставки правит техник без программиста |
| Расчёт в Excel → клиенту в ватсап руками | Расчёт в 2 клика → красивый PDF или сообщение в WhatsApp прямо из CRM |
| Менеджеры забывают клиентов | «Требует внимания» на дашборде, метка «⚠ без задачи» в канбане, авто-задача при смене этапа |
| Новичок не знает, что делать и что говорить | 7 маршрутов сопровождения по типу договора: на каждом этапе чек-лист, «что делать» и готовый скрипт «что сказать клиенту» |
| Непонятно, где теряются клиенты | Воронка конверсии, источники, причины закрытия — считаются по всей базе |
| Ватсап-переписка в личных телефонах | Встроенный WhatsApp (Green API): общий инбокс, шаблоны, автоответ новым лидам, anti-ban лимиты |
| Документы клиентов в чатах | Google Drive: файлы по клиентам и этапам автоматически |

## Кому продаём (сегменты, в порядке приоритета)

### 1. Ипотечные брокеры / агентства РК (2–15 менеджеров) — ядро
- **Боль:** лиды из Instagram/TikTok теряются, менеджеры считают в Excel, руководитель не видит воронку.
- **Кто платит:** владелец агентства. **Кто пользуется:** менеджеры, спец по сопровождению.
- **Питч:** «Ваши менеджеры закрывают больше сделок с тех же лидов: система сама говорит, кому позвонить и что сказать».

### 2. Риелторские агентства с ипотечным направлением
- **Боль:** риелтор умеет продавать квартиры, но плавает в госпрограммах и одобрении.
- **Питч:** «Риелтор без ипотечного опыта считает Наурыз и Отбасы как профи — маршрут и скрипты внутри».
- Отдельная ценность: маршрут «Поиск дома» — подбор, просмотры, юр. чистота, задаток.

### 3. Отделы продаж застройщиков / ЖК
- **Боль:** покупатель уходит «думать про ипотеку» и не возвращается.
- **Питч:** «Менеджер ЖК за 2 минуты показывает клиенту платёж по всем госпрограммам и сразу ставит его на сопровождение — сделка не уходит».

### 4. Независимые ипотечные консультанты (соло)
- **Боль:** всё в голове и в заметках телефона.
- **Питч:** «Личный ассистент за копейки: не забудешь ни одного клиента, PDF-расчёты с твоим именем».
- Вход в продукт: дёшево/бесплатно → апсейл при росте до команды.

## Роли внутри продукта (готово в системе)

| Роль | Что видит/делает |
|---|---|
| **Менеджер** | свои лиды, канбан, WhatsApp, калькулятор, задачи, «следующий шаг» |
| **Специалист** | клиенты на сопровождении: маршрут этапов, чек-листы, документы |
| **Руководитель** | все клиенты, KPI: воронка, источники, причины закрытия, выручка по менеджерам |
| **Техник** | ставки программ, чек-листы этапов, шаблоны WA, пользователи |

## Ключевые фичи для питча (демо-сценарий, 7 минут)

1. Пишет «клиент» в WhatsApp → лид сам появился в CRM + автоответ.
2. Менеджер открывает чат → «Рассчитать → в сообщение» → отправил расчёт за 30 секунд.
3. Карточка клиента → калькулятор → «Сохранить в карточку» + «PDF клиенту».
4. Перевели в «Договор» → CRM сама создала задачу «Отправить договор».
5. Вкладка «Сопровождение»: маршрут под программу клиента, на этапе — что делать + скрипт «что сказать» (кнопка Копировать).
6. Канбан: у кого «⚠ без задачи» — сразу видно, кого забыли.
7. KPI: воронка + «почему теряем клиентов» (причины закрытия).

## Возражения

- **«У нас уже amoCRM»** — amo остаётся для маркетинга, у нас — ипотечная механика: программы РК, расчёты, маршруты сопровождения. Либо переезд: у нас дешевле и заточено под вас.
- **«Менеджеры не будут заполнять»** — большинство заполняется само: лиды из WA, авто-задачи, авто-приветствие. Менеджеру остаётся ставить галочки.
- **«Ставки меняются»** — техник правит ставки и программы в админке за минуту, без разработчика.
- **«Данные клиентов — безопасно?»** — своя база (Supabase), доступ только через сервер, роли, rate-limit, история ошибок. Можно развернуть отдельную инсталляцию только под вас.

## Модель продажи (гипотеза, пока отдельные инсталляции)

| Тариф | Кому | Что входит | Цена (гипотеза) |
|---|---|---|---|
| **Соло** | консультант | 1 пользователь, WA, калькулятор | ~15 000 ₸/мес |
| **Команда** | агентство до 5 | роли, KPI, Drive, автоответ | ~49 000 ₸/мес |
| **Агентство** | 5–15 + | всё + отдельная инсталляция, поддержка, обучение | ~99 000 ₸/мес + сетап |

Сетап-фи за развёртывание (Supabase + Vercel + Green API + миграции) — разовая, ~50–100 тыс. ₸.
Решение «SaaS vs инсталляции» отложено: пока продаём инсталляции (безопаснее, нет мультитенантности).

## Каналы (первые шаги)

1. **Свои кейсы** — скриншоты воронки/KPI своего агентства: «как мы ведём N клиентов».
2. Instagram/TikTok для брокеров: «как считать Наурыз за 30 секунд» (демо калькулятора).
3. Чаты ипотечных брокеров и риелторов (WhatsApp/Telegram).
4. Партнёрка с обучающими курсами для ипотечных брокеров: CRM в комплекте курса.
5. Демо-стенд: тестовый логин с фейковыми данными — показывать вживую.
