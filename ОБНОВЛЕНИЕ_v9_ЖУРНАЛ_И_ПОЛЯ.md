## 1. Журнал действий 📜

Панель техника → вкладка **«Журнал»** (только техник/руководитель):
- Кто что сделал: создание, правка, смена этапа, удаление клиента, изменение воронки.
- Иконка+цвет по типу действия, автор, деталь (например «analysis → contract»), время.
- Последние 200 записей, кнопка «Обновить».

Пишется автоматически на сервере (`lib/actionLog.js`, fire-and-forget — журнал
никогда не роняет основную операцию). Точки логирования:
`POST/PUT/DELETE /api/clients`, `PUT /api/pipeline`.

## 2. Конструктор полей карточки 🧩

Панель техника → вкладка **«Поля карточки»** (техник/руководитель):
- Добавьте свои поля под процесс: текст / число / дата / список (варианты через запятую).
- Примеры: «Ставка банка», «Дата встречи», «Ответственный юрист».

В карточке клиента (вкладка **Клиент** → секция «Дополнительные поля») эти поля
появляются автоматически, значения сохраняются в клиенте (автосейв, как всё остальное).
Конфигурация полей отдаётся всем ролям через `/api/calc` (calc-settings закрыт ролями — не трогали).
